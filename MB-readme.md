# 道威Vue3项目 - 移动端

## 简介
道威Vue3项目的移动端(MP)是针对移动设备优化的轻量级界面，提供关键功能的便捷访问。移动端应用采用触控友好的界面设计，适合随时随地进行基本业务操作和管理任务。

## 技术实现
- **前端框架**: Vue 3
- **UI组件库**: Vant
- **状态管理**: Pinia
- **布局**: 移动优先设计，适配各种屏幕尺寸

## 系统模块

### 公共组件
- **头部**: 显示当前页面信息和必要操作按钮
- **底部导航**: 提供主要功能区域的快速访问
- **主布局**: 灵活的页面容器，优化移动体验
- **首页**: 系统关键信息概览
- **支持页面**: 帮助与支持信息

### 登录模块
- **用户登录**: 普通用户的移动登录入口
- **管理员登录**: 管理员的移动登录入口

### 管理员模块
- **用户管理**: 管理系统用户账户
- **客户管理**: 客户信息的简化管理
- **订单管理**: 订单状态查看和基本操作
- **样品管理**: 样品信息的查询和简单管理
- **员工管理**: 员工账户的基本管理功能

### 用户模块
- **客户管理**: 移动端的客户信息管理
- **订单管理**: 移动端的订单处理
- **样品管理**: 移动端的样品管理

### 工具箱
- **图像压缩**: 手机拍摄图片的优化
- **图像文字识别**: 移动场景下的OCR功能
- **ZIP文件处理**: 简化的ZIP文件创建与解压
- **AI工具**: 适合移动端的AI辅助功能

## 移动端特点

1. **触控优化**: 针对触摸屏设计的界面元素和交互
2. **轻量级UI**: 减少不必要的视觉元素，提高性能
3. **离线功能**: 部分功能支持弱网或离线环境使用
4. **自适应布局**: 适配各种移动设备尺寸
5. **手势操作**: 支持常见移动手势操作
6. **专注核心任务**: 简化工作流程，专注最常用功能

## Vant组件集成
移动端整合了以下Vant组件以提供优质的移动体验:
- Button, Cell, Form等基础组件
- Dialog, Toast等反馈组件
- List, PullRefresh等数据展示组件
- Uploader, Search等表单组件
- DatePicker, Tabs等高级组件

## 使用说明

### 系统入口
通过 `/mp.html` 访问移动端应用

### 开发模式
```bash
npm run dev
```
然后在浏览器中访问 `http://localhost:3000/mp.html`（可使用浏览器的移动设备模拟模式）

### 真机测试
确保开发服务器在局域网内可访问，然后通过移动设备浏览器访问对应IP和端口。

### 构建部署
```bash
npm run build
```
构建结果位于 `dist` 目录，可部署到任意静态Web服务器

## 响应式设计

移动端应用通过CSS媒体查询和弹性布局实现了优秀的响应式设计:
- 针对手机设备(小于768px)优化
- 适配平板设备(768px-992px)
- 流畅处理屏幕方向变化
