# 道威Vue3项目 - PC端

## 简介
道威Vue3项目的PC端是针对桌面环境优化的完整界面，提供全面的功能和高效的操作体验。PC端采用现代化的界面设计，适合在办公环境中进行复杂的业务操作和管理任务。

## 技术实现
- **前端框架**: Vue 3 + TypeScript
- **UI组件库**: Element Plus
- **状态管理**: Pinia
- **布局**: 响应式设计，适配不同分辨率屏幕

## 系统模块

### 公共组件
- **Header**: 顶部导航栏，显示用户信息和全局操作
- **Menu**: 侧边菜单，根据用户角色动态显示可访问功能
- **Home**: 系统首页，展示概览信息
- **Support**: 帮助支持页面

### 登录模块
- **用户登录**: 普通用户的登录入口
- **管理员登录**: 管理员的登录入口，具有更高权限

### 管理员模块
- **仪表盘**: 系统数据概览和可视化统计
- **用户管理**: 管理系统用户账户和权限
- **客户管理**: 客户信息的管理与维护
- **订单管理**: 订单流程的全生命周期管理
- **样品管理**: 样品信息的录入、查询和管理
- **员工管理**: 系统内部员工账户和权限管理

### 用户模块
- **客户管理**: 用户视角的客户信息管理
- **订单管理**: 用户视角的订单处理
- **样品管理**: 用户视角的样品管理

### 工具箱
- **图像压缩**: 优化图片大小和质量
- **图像文字识别**: OCR功能，提取图片中的文字
- **ZIP文件创建**: 创建加密或非加密的ZIP压缩文件
- **ZIP文件解压**: 解压ZIP文件并处理内容
- **AI工具集**: 集成多种AI辅助功能

## 路由结构
PC端路由基于角色和模块进行组织，主要包括:
- `/home`: 系统首页
- `/login`和`/admin/login`: 登录入口
- `/support`: 支持页面
- `/admin/*`: 管理员功能
- `/user/*`: 用户功能
- `/tools/*`: 工具集功能

## 特性亮点
1. **响应式界面**: 适配不同尺寸的显示设备
2. **权限控制**: 基于角色的精细化权限管理
3. **数据可视化**: 采用ECharts进行数据图表展示
4. **丰富组件**: 集成多种高级UI组件
5. **优化性能**: 合理的懒加载和缓存策略

## 使用说明

### 系统入口
通过 `/pc.html` 访问PC端应用

### 开发模式
```bash
npm run dev
```
然后在浏览器中访问 `http://localhost:3000/pc.html`

### 构建部署
```bash
npm run build
```
构建结果位于 `dist` 目录，可部署到任意静态Web服务器
